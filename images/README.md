# Images

Place the following image files in this directory before deploying:

| File | Description | Recommended size |
|------|-------------|-----------------|
| `flower.png` | Floral corner decoration (top-left & top-right of Page 1). A transparent-background PNG works best. | 300 × 400 px |
| `banana-leaf.png` | Banana leaf corner decoration (top-left & top-right of Page 2). Transparent PNG preferred. | 250 × 500 px |
| `ghibli-couple.png` | Couple illustration shown in the circular frame on Page 3. A Studio Ghibli–style or watercolour portrait works beautifully. | 500 × 500 px (square) |

> **Tip:** If an image file is missing the site will still load – the `onerror` handlers in `index.html` hide broken images gracefully, and an emoji fallback is shown for the couple photo.
